十进制转二进制

整数部分转二进制

  • 除2取余法
  • 说明：整数部分除以2,取余数,商继续除以2,直到商为0,排列余数即为二进制
计算 商 余数
9 % 2 4 1
4 % 2 2 0
2 % 2 1 0
1 % 2 0 1
  • 9(十进制) = 1001(二进制)

小数部分转二进制

  • 乘2取整法
  • 说明：小数部分乘以2,取整数，小数部分继续乘以2,直到小数部分为0,排列整数即为二进制
计算 整数 小数
0.9 * 2 1 0.8
0.8 * 2 1 0.6
0.6 * 2 1 0.2
0.2 * 2 0 0.4
0.4 * 2 0 0.8
  • 1100开始无限循环：0.9(十进制) = 0.1(1100)(二进制）

二进制转十进制

  • 二进制数非零位取2的索引位次幂累加,整数部分与小数部分分别累加
  • 比如二进制数1001.1110
    | 二进制位 | 索引位 | 值|
    | :————- | :————- |:————- |
    | 1 | 3 | 2^3=8 |
    | 0 | 2 | 位0不取 |
    | 0 | 1 | 位0不取 |
    | 1 | 0 | 2^0=1 |
    | 1 | -1 | 2^-1=0.5 |
    | 1 | -2 | 2^-2=0.25 |
    | 1 | -3 | 2^-3=0.125 |
    | 0 | -4 | 位0不取 |
  • 整数部分8+1=9,小数部分0.5+0.25+0.125 = 0.875 结果为9.875
